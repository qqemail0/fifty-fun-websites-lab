## 创作灵感

偏创作与表达，不再只是卡片生成，而是故事、诗、标题、梦境和选题的独立工具。

- [灵感锻造炉](https://qqemail0.github.io/fifty-fun-websites-lab/sites/idea-forge/): 输入碎片词，锻造成可执行选题和开场句。，工具类型 `mixer`，视觉 `editorial`
- [故事神谕台](https://qqemail0.github.io/fifty-fun-websites-lab/sites/story-oracle/): 三列抽签生成角色、欲望和转折，适合短故事开局。，工具类型 `slot`，视觉 `theatre`
- [梦境索引馆](https://qqemail0.github.io/fifty-fun-websites-lab/sites/dream-index/): 像档案馆一样保存梦境碎片，自动标记色彩和隐喻。，工具类型 `archive`，视觉 `museum`
- [颜色诗生成器](https://qqemail0.github.io/fifty-fun-websites-lab/sites/color-poem/): 点击色块生成短诗，并把颜色转成情绪句子。，工具类型 `poem`，视觉 `soft`
- [爆款标题实验室](https://qqemail0.github.io/fifty-fun-websites-lab/sites/title-lab/): 输入标题，实时获得钩子、清晰度和点击欲评分。，工具类型 `scorer`，视觉 `studio`
